    Juego - Dinosaurio de Google

    Visitar proyecto
    Juego de navegador creado para entretener, el objetivo principal de este juego es ayudar al dinosaurio a evitar obstáculos y sobrevivir el mayor tiempo posible sin chocar con ellos. Los obstáculos principales incluyen cactus y, a medida que el juego avanza, también aparecen pterodáctilos.

    Tecnologías utilizadas:

    Html5

    CSS3

    JavaScript
